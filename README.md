# RFM Customer Segmentation Analysis Using BigQuery

## Project Overview
This project uses SQL in Google BigQuery to perform ETL processing and customer segmentation analysis using the RFM (Recency, Frequency, Monetary) framework.

The analysis was designed to identify customer purchasing behaviour, segment customers based on engagement and value, and support data-driven business decision-making.

---

# Objectives
- Perform ETL and data transformation using SQL
- Analyse customer transaction behaviour
- Segment customers using the RFM framework
- Identify high-value and at-risk customers
- Generate actionable business insights

---

# Tools & Technologies
- Google BigQuery
- SQL
- Microsoft Excel
- Data Cleaning
- Customer Analytics
- Business Intelligence

---

# Workflow
1. Data extraction and loading
2. SQL-based ETL and transformation
3. RFM metric calculation
4. Customer segmentation
5. Insight generation and reporting

---

# Key Insights
- Identified loyal and high-value customers
- Highlighted inactive and at-risk customer groups
- Analysed purchasing frequency and spending behaviour
- Supported customer retention and marketing strategies

---

# Business Impact
This project demonstrates how SQL-based analytics and customer segmentation can help businesses improve retention strategies, optimise marketing efforts, and better understand customer value.
